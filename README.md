# HKDF-BLAKE2b

[HKDF](https://www.rfc-editor.org/rfc/rfc5869) with [keyed BLAKE2b-512](https://www.rfc-editor.org/rfc/rfc7693) instead of [HMAC](https://www.rfc-editor.org/rfc/rfc2104). More efficient but non-standard.
